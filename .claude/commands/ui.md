## 设计规范参考

### 本项目技术栈
- React + Next.js 15
- Ant Design + Tailwind CSS
- Framer Motion 动效

### 设计风格
- Glassmorphism 毛玻璃效果
- 圆角 xl/2xl
- 紫色渐变主色调
- 微交互动效

### 常用操作
- `plan` - 设计规划
- `build` - 构建组件
- `create` - 创建新页面/组件
- `review` - 审查 UI 代码
- `fix` - 修复样式问题
- `improve` - 优化提升
